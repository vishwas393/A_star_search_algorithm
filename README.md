# A * Search Algorithm


This is a A* Search Algorithm implementation in C++. 
Run :
```sh
cd /path/to/project_directory/
./run_a_star_search_algo
```
As an example input values, enter
 - Start Point X: 2
 - Start Point Y: 2
 - End Point X: 5
 - End Point Y: 46
